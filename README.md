
# Python Programming Q&A Assistant

AI-powered Python Q&A Assistant built using Retrieval-Augmented Generation (RAG), FastAPI, and the Stack Overflow Python Questions & Answers dataset.

## Features
- RAG-based question answering
- FastAPI REST API
- Vector search using FAISS
- Grounded responses from Stack Overflow data
- API testing with Pytest

## Dataset
Stack Overflow Python Questions & Answers Dataset

## API Endpoints

### Health Check
GET /health

### Ask Question
POST /ask

## Tech Stack
- Python
- FastAPI
- LangChain
- FAISS
- Sentence Transformers
- Gemini/OpenAI
